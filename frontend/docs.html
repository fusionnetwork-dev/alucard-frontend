Alucard Documentation

Alucard is a free, open-source Lua script protection and HWID whitelist system built on Cloudflare Workers for maximum speed, security, and reliability.

🚀 Base URL: https://api.alucard.lol

How Alucard Works

Alucard uses a simple but powerful workflow to protect your Lua scripts:

  1. Upload Your Script: You upload your Lua script through the dashboard
  2. Get Your Loader: Alucard generates a unique URL for your script (e.g., api.alucard.lol/s/abc123)
  3. Automatic Protection: Your script is automatically obfuscated and encrypted
  4. HWID Verification: When someone runs your script, Alucard checks if their HWID is whitelisted
  5. Secure Delivery: If verified, the protected script is loaded; otherwise, access is denied
✅ Key Benefits:
  • No complex setup - just upload and use
  • Automatic script obfuscation with military-grade encryption
  • HWID-based access control
  • Real-time analytics and webhook notifications
  • 7-day whitelist expiration to prevent key sharing

HWID System Explained

Alucard uses Hardware IDs (HWIDs) to control who can access your scripts:

What is an HWID?

A Hardware ID is a unique identifier for a device. Alucard uses the format: ALUCARD-XXXXXXXXXXXX

How to Get Whitelisted

  1. Visit alucard.lol/unlock
  2. Complete 3 quick ad checkpoints (takes ~45 seconds)
  3. Enter your ALUCARD key
  4. Your device is whitelisted for 7 days
⚠️ Important:
  • Each HWID expires after 7 days and must be renewed
  • Maximum 3 unlocks per 24 hours per IP/user
  • Each HWID can only be registered to one account

Script Protection

When you upload a script to Alucard, it's automatically protected with multiple layers of security:

Protection Layers

  1. Variable Renaming: All variable names are replaced with confusing alternatives
  2. String Encryption: Strings are encrypted using XOR cipher with random keys
  3. Control Flow Obfuscation: Adds junk code and fake conditionals
  4. Anti-Debug: Detects decompilers and debugging tools
  5. AES-256 Encryption: Final layer of military-grade encryption

Quick Start Guide

Get up and running in under 5 minutes:

Step 1: Create an Account

POST /api/signup
Content-Type: application/json

{
  "email": "you@example.com",
  "username": "yourname",
  "password": "secure_password"
}

Step 2: Upload Your Script

Go to your dashboard and upload your Lua script. You'll receive a unique loader URL.

Step 3: Use the Loader in Roblox

-- Replace YOUR_HWID with your actual HWID
loadstring(game:HttpGet("https://api.alucard.lol/s/YOUR_SCRIPT_ID?hwid=YOUR_HWID"))()

Step 4: Get Your HWID Whitelisted

Visit alucard.lol/unlock and complete the quick verification process.

🎉 That's it! Your script is now protected and will only run for whitelisted HWIDs.

Script Loader (Public Endpoint)

GET /s/:scriptId

Load a protected script. This is the main endpoint used in Roblox scripts.

Query Parameters

Parameter Type Description
hwid string The Hardware ID to verify (required)

How It Works

  1. Receives request with script ID and HWID
  2. Checks if HWID is whitelisted and active
  3. Verifies HWID hasn't expired (7-day limit)
  4. Returns obfuscated script code if authorized
  5. Logs access and increments download counter
  6. Sends webhook notification (if configured)

Example Usage

-- In your Roblox script
loadstring(game:HttpGet("https://api.alucard.lol/s/abc123?hwid=ALUCARD-ABC123XYZ"))()

Possible Responses

-- Success: Returns protected Lua code
-- Error: Access denied: HWID not whitelisted
-- Error: Access denied: HWID expired (7 day limit)
-- Error: Script not found

Authentication

Most management endpoints require a Bearer token in the Authorization header.

Authorization: Bearer YOUR_TOKEN_HERE

You receive this token when you login or signup. Store it securely and include it in all authenticated requests.

Whitelist Management

POST /api/unlock-whitelist

Whitelist a device after completing ad checkpoints.

Headers

Authorization: Bearer YOUR_TOKEN

Request Body

Parameter Type Description
hwid string Hardware ID (format: ALUCARD-XXXX)
deviceType string Device type (e.g., "Windows", "Mobile")
Rate Limits: Maximum 3 unlocks per 24 hours per IP and per user account.
POST /api/verify

Verify if an HWID is currently whitelisted and active.

Headers

Authorization: Bearer YOUR_TOKEN

Request Body

Parameter Type Description
hwid string Hardware ID to check

Response

{
  "verified": true,
  "message": "Verified successfully",
  "expiresIn": 5  // days remaining
}

Get all scripts for the authenticated user.

Headers

Delete a script permanently.

Roblox Integration

Here's how to use Alucard in your Roblox scripts:

Basic Example

-- Load your protected script
loadstring(game:HttpGet("https://api.alucard.lol/s/YOUR_SCRIPT_ID?hwid=YOUR_HWID"))()

With Error Handling

local success, result = pcall(function()
    return loadstring(game:HttpGet("https://api.alucard.lol/s/abc123?hwid=ALUCARD-XYZ"))()
end)

if not success then
    game.Players.LocalPlayer:Kick("Verification failed: " .. tostring(result))
    return
end

print("✅ Script loaded successfully!")
-- Your code continues here

Getting HWID Dynamically (Advanced)

-- Get HWID from game (example for Synapse/Krnl)
local hwid = game:GetService("RbxAnalyticsService"):GetClientId()

-- Or use a custom HWID function
local function getHWID()
    return "ALUCARD-" .. game:GetService("Players").LocalPlayer.UserId
end

local myHWID = getHWID()
local scriptUrl = "https://api.alucard.lol/s/abc123?hwid=" .. myHWID

loadstring(game:HttpGet(scriptUrl))()

Complete Examples

JavaScript/Node.js Client

class AlucardClient {
    constructor(apiUrl = 'https://api.alucard.lol') {
        this.apiUrl = apiUrl;
        this.token = null;
    }

    async login(email, password) {
        const res = await fetch(`${this.apiUrl}/api/login`, {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({ email, password })
        });
        const data = await res.json();
        this.token = data.token;
        return data;
    }

    async uploadScript(name, content, description) {
        const res = await fetch(`${this.apiUrl}/api/upload-script`, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
                'Authorization': `Bearer ${this.token}`
            },
            body: JSON.stringify({ name, content, description })
        });
        return await res.json();
    }

    async getScripts() {
        const res = await fetch(`${this.apiUrl}/api/dashboard/scripts`, {
            headers: { 'Authorization': `Bearer ${this.token}` }
        });
        return await res.json();
    }
}

// Usage
const client = new AlucardClient();
await client.login('you@example.com', 'password');

const result = await client.uploadScript(
    'My Script',
    'print("Hello World")',
    'A simple test script'
);

console.log('Script URL:', result.scriptUrl);
console.log('Loader:', result.loader);

Python Client

import requests

class AlucardClient:
    def __init__(self, api_url="https://api.alucard.lol"):
        self.api_url = api_url
        self.token = None
    
    def login(self, email, password):
        response = requests.post(
            f"{self.api_url}/api/login",
            json={"email": email, "password": password}
        )
        data = response.json()
        self.token = data["token"]
        return data
    
    def upload_script(self, name, content, description):
        response = requests.post(
            f"{self.api_url}/api/upload-script",
            headers={"Authorization": f"Bearer {self.token}"},
            json={
                "name": name,
                "content": content,
                "description": description
            }
        )
        return response.json()
    
    def get_scripts(self):
        response = requests.get(
            f"{self.api_url}/api/dashboard/scripts",
            headers={"Authorization": f"Bearer {self.token}"}
        )
        return response.json()

# Usage
client = AlucardClient()
client.login("you@example.com", "password")

result = client.upload_script(
    "My Script",
    'print("Hello World")',
    "A simple test script"
)

print(f"Script URL: {result['scriptUrl']}")
print(f"Loader: {result['loader']}")

Rate Limits & Security

Rate Limits

Endpoint Type Limit
Script Loading (/s/:id) Unlimited
Authentication 10 requests/minute
Whitelist Unlock 3 per 24 hours (per IP & user)
API Management 100 requests/hour

Security Features

  • SHA-256 Hashing: All passwords and HWIDs are hashed
  • Bypass Detection: Automatic flagging of suspicious behavior
  • IP Logging: All requests are logged for security audits
  • 7-Day Expiration: HWIDs expire to prevent key sharing
  • Cloudflare Protection: Built-in DDoS protection
⚠️ Bypass Detection: Alucard monitors for VPN usage, rapid requests, suspicious user agents, and other bypass attempts. Violators are automatically flagged and may be blocked.
