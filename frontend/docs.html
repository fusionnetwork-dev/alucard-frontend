Documentation

Welcome to the Alucard API documentation. Alucard is a free, open-source Lua script protection and whitelist system built on Cloudflare Workers for maximum speed and reliability.

Base URL: https://api.alucard.lol

Quick Start

Get up and running in under 5 minutes:

Step 1: Create an Account

POST /api/signup
Content-Type: application/json

{
  "email": "you@example.com",
  "username": "yourname",
  "password": "secure_password"
}

Step 2: Get Your HWID Whitelisted

Visit alucard.lol/unlock and complete the 3 quick steps to whitelist your device.

Step 3: Upload Your Script

Go to your dashboard and upload your Lua script. We'll automatically protect it with encryption and obfuscation.

Step 4: Integrate

-- In your Roblox script
loadstring(game:HttpGet("https://api.alucard.lol/verify"))()

-- Your protected code runs after verification
print("User is whitelisted!")

Authentication

Most endpoints require a Bearer token in the Authorization header. You receive this token when you login or signup.

Authorization: Bearer YOUR_TOKEN_HERE

Whitelist Endpoints

POST /api/unlock-whitelist

Whitelist a device after completing ad checkpoints.

Request Body

Parameter Type Description
userId string User ID
hwid string Hardware ID to whitelist
Note: User must complete all 3 ad checkpoints before calling this endpoint.
POST /api/verify

Verify if a HWID is whitelisted.

Request Body

Parameter Type Description
hwid string Hardware ID to check

Response

{
  "verified": true,
  "message": "Verified successfully"
}
POST /api/blacklist

Blacklist a device (requires authentication).

Headers

Authorization: Bearer YOUR_TOKEN

Request Body

Parameter Type Description
hwid string Hardware ID to blacklist

Roblox Integration

Here's how to integrate Alucard into your Roblox scripts:

Basic Integration

-- Load the whitelist verification
loadstring(game:HttpGet("https://api.alucard.lol/verify"))()

-- Your script code here (only runs if verified)
print("User is whitelisted!")
game.Players.LocalPlayer:Kick("Script loaded successfully!")

With Error Handling

local success, err = pcall(function()
    loadstring(game:HttpGet("https://api.alucard.lol/verify"))()
end)

if not success then
    game.Players.LocalPlayer:Kick("Verification failed: " .. tostring(err))
    return
end

-- Your protected code
print("Access granted!")

Web Integration

Using Alucard in web applications:

// Login
const response = await fetch('https://api.alucard.lol/api/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    email: 'you@example.com',
    password: 'your_password'
  })
});

const { token } = await response.json();
localStorage.setItem('token', token);

// Use the token for authenticated requests
const scripts = await fetch('https://api.alucard.lol/api/scripts', {
  headers: { 'Authorization': `Bearer ${token}` }
});

Code Examples

JavaScript/Node.js

const AlucardAPI = {
  baseURL: 'https://api.alucard.lol',
  token: null,

  async login(email, password) {
    const res = await fetch(`${this.baseURL}/api/login`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ email, password })
    });
    const data = await res.json();
    this.token = data.token;
    return data;
  },

  async verifyHWID(hwid) {
    const res = await fetch(`${this.baseURL}/api/verify`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ hwid })
    });
    return await res.json();
  }
};

// Usage
await AlucardAPI.login('you@example.com', 'password');
const verified = await AlucardAPI.verifyHWID('HWID-ABC123');

Python

import requests

class AlucardAPI:
    BASE_URL = "https://api.alucard.lol"
    
    def __init__(self):
        self.token = None
    
    def login(self, email, password):
        response = requests.post(
            f"{self.BASE_URL}/api/login",
            json={"email": email, "password": password}
        )
        data = response.json()
        self.token = data["token"]
        return data
    
    def verify_hwid(self, hwid):
        response = requests.post(
            f"{self.BASE_URL}/api/verify",
            json={"hwid": hwid}
        )
        return response.json()

# Usage
api = AlucardAPI()
api.login("you@example.com", "password")
result = api.verify_hwid("HWID-ABC123")

Rate Limits

To ensure fair usage, the following rate limits apply:

  • Authentication endpoints: 10 requests per minute
  • Verification endpoints: 100 requests per minute
  • API endpoints: 1000 requests per hour per API key
Bypass Detection: Alucard has built-in bypass detection. VPNs, rapid requests, and suspicious behavior will be flagged and may result in automatic blocking.

Security

Alucard takes security seriously:

  • All passwords are hashed using SHA-256
  • HWIDs are hashed before storage
  • Rate limiting on all endpoints
  • IP logging for security audits
  • Automatic flagging of suspicious activity
  • DDoS protection via Cloudflare
